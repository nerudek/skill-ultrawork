# ultrawork

Parallel execution engine - run multiple tasks simultaneously for speed

## Installation

```bash
# Clone to your OpenClaw skills directory
git clone https://github.com/nerudek/skill-ultrawork.git ~/.openclaw/skills/ultrawork

# Or symlink if you keep skills elsewhere
ln -s /path/to/skill-ultrawork ~/.openclaw/skills/ultrawork
```
